# Examensarbete hos Soleil
En React Native app som hämtar data med hjälp av SiteVision Headless.
Tanken är att appen ska fungera dynamiskt där man endast ska behöva ändra de olika URL:er
som vi fetchar ifrån så ska den visa data från just den webbplatsen som är byggd med SiteVision.

Projektet är inte färdigt ännu.
